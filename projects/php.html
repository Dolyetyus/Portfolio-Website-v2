Chat Application Backend

This project is a backend service for a chat application, as requested by Bunq, providing RESTful APIs for user management, group, and messaging functionalities.

The backend is built using PHP with the Slim framework, along with SQLite for data storage. Below is a brief overview of the project:

Features

  • User registration and deletion
  • Group creation, joining, and deletion
  • Messaging within groups
  • Retrieve group members and messages
  • Edge case handling

API Endpoints

The available API endpoints are as follows:

  • User Registration: (POST /users/register)
  • User Deletion: (DELETE /users/{user_id})
  • Group Creation: (POST /groups/create)
  • Group Join: (POST /groups/join)
  • Group Deletion: (DELETE /groups/{group_id})
  • Get Group Members: (GET /groups/{group_id}/members)
  • Group Leave: (POST /groups/leave)
  • Send Message: (POST /groups/{group_id}/messages/send)
  • Get Group Messages: (GET /groups/{group_id}/messages)

For a full list of API requests, please refer to the official documentation on the repository (private on Bunq's request).
