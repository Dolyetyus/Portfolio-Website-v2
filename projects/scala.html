Reversible Snake Game

Made a dynamic Snake game that integrates Object-Oriented Programming and immutability for in-depth game state management. Through an implementation using immutable data structures, the project ensures integrity and consistency throughout gameplay. The architecture supports interactive features like frame-by-frame reversal, facilitated by a history of past game states and activation of reverse mode when pressed 'R'. The game accurately reflects user inputs, updates the state, and renders visuals. To optimize performance and adhere to functional programming principles, mutable containers are not used in favor of immutable data structures, enhancing reliability, maintainability, and best practices for such project.
