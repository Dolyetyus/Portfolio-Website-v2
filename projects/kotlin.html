TaskTogether - Collaborative Task Management App

TaskTogether is a Kotlin-based Android app designed to help users manage and collaborate on tasks with a partner or friend. It offers features like task assignment, real-time updates, task sorting, and Firebase integration for secure user authentication and real-time task management.

Features

  • Collaborative task management with real-time updates.
  • Task assignment to yourself, your partner, or both.
  • Sort tasks by title, due date, or priority.
  • Search tasks by title.
  • Partner integration for task collaboration.
  • Firebase Authentication and Realtime Database integration.

Project Highlights

  • Firebase Authentication for user registration and login.
  • Real-time task updates with Firebase Realtime Database.
  • RecyclerView and custom adapters for dynamic task management.
  • Search functionality with SearchView.
  • Over 3000 lines of Kotlin code and 5000+ lines overall and 10+ classes, focusing on object oriented and functional programming.
